#  ESP32 Knight Rider LED Project

##  Overview

This project is my first embedded systems project using the ESP32.  
It demonstrates how to control multiple LEDs using GPIO pins and create a Knight Rider (KITT) scanning effect.

---

##  Hardware Used

- ESP32 Development Board  
- 5 x LEDs  
- 5 x Resistors (220Ω)  
- Breadboard  
- Jumper wires  
- USB cable  

---

##  GPIO Pins Used

- GPIO 2  
- GPIO 4  
- GPIO 5  
- GPIO 18  
- GPIO 19  

Each LED is connected through a resistor to protect the circuit.

---

## Software

- ESP-IDF  
- C Programming Language  
- FreeRTOS (`vTaskDelay`)  

---

## How It Works

- LEDs turn ON one by one  
- Then reverse direction  
- This creates a scanning light effect
